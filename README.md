# אתר אדירכלות רות גצל

ברוכים הבאים לפרויקט אתר אדירכלות של רות גצל!

## מבנה הפרויקט
- `RutGetsel.html` — דף הבית של האתר
- `style.css` — קובץ עיצוב (CSS)
- `script.js` — קובץ סקריפטים (JavaScript)
- `FinalProject/` — תיקיית פרויקט סופי
- `kindergarten/` — תיקיית גן ילדים
- `kitchen/` — תיקיית מטבח

## איך להפעיל
1. פתחו את הקובץ `RutGetsel.html` בדפדפן.
2. ודאו שכל הקבצים הנלווים (CSS, JS) נמצאים באותה תיקייה.

## טכנולוגיות
- HTML
- CSS
- JavaScript

## יוצרת האתר
רות גצל
